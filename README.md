# Abiotic Factor 游戏攻略网站

🎮 **项目概述**  
为《Abiotic Factor》游戏打造的专业官方网站，提供游戏信息、攻略指南、社区交流等功能。

---

## 📋 项目信息

- **域名**: https://abioticfactor.net
- **项目状态**: 规划阶段
- **开发团队**: 高级前端工程师 + 项目负责人
- **目标用户**: Abiotic Factor游戏玩家、潜在玩家

---

## 🛠️ 技术栈规划

### 前端技术
- **框架**: React.js / Next.js (考虑SEO优化)
- **样式**: Tailwind CSS + 自定义组件
- **动画**: Framer Motion (游戏感交互)
- **响应式**: 移动端优先设计

### SEO优化
- **结构化数据**: JSON-LD标记
- **页面速度**: 图片优化、代码分割
- **内容优化**: 游戏关键词布局
- **社交媒体**: Open Graph、Twitter Cards

### 性能优化
- **图片优化**: WebP格式、懒加载
- **代码优化**: Tree shaking、代码压缩
- **CDN**: 静态资源分发

## 📝 讨论记录

### 需求讨论

#### 项目最终需求确认 ✅
**网站类型**: 单页面《Abiotic Factor》专题攻略网站
**域名**: https://abioticfactor.net
**参考设计**: 游戏官网 https://www.abioticfactor.com/

#### 核心要求
1. **设计风格**: 与官网保持一致，体现潮流、游戏、二次元、生存游戏元素
2. **单页面设计**: 所有内容集中在首页，滚动浏览
3. **多语言支持**: 11种语言（默认英文）
4. **响应式设计**: PC、移动端、平板三端适配
5. **技术选型**: 纯HTML/CSS/JS，无后端数据库
6. **部署方式**: GitHub + Cloudflare

#### 功能模块
- 游戏介绍与预告片
- 主流程攻略 (Main Process)
- 详细攻略 (Detailed Walkthrough)  
- 地图指南 (Guide Map)
- 用户评价 (Valuation)
- 多语言切换
